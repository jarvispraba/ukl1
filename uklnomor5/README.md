# Program Menghitung Nilai Rata-Rata Ujian Produktif RPL

## Deskripsi
Program ini dibuat untuk membantu seorang pendidik dalam melakukan rekapitulasi nilai ujian Produktif RPL
Dengan menggunakan program ini, guru dapat memasukkan sejumlah nilai siswa, kemudian program akan menghitung rata-rata nilai ujian secara otomatis

## Cara Kerja Program
1. Pengguna memasukkan jumlah siswa terlebih dahulu.  
2. Program akan meminta pengguna untuk memasukkan nilai ujian sebanyak jumlah siswa tersebut.  
3. Setelah semua nilai dimasukkan, program menghitung rata-rata dari seluruh nilai.  
4. Hasil rata-rata akan ditampilkan di layar.

## Rumus Rata-Rata
Rata-rata = (Jumlah seluruh nilai) / (Jumlah siswa)
